## 🔥 One-Minute Quick Start

### Step 1: Check if Node.js installed
```cmd
node --version
```
If error → Install from https://nodejs.org/

### Step 2: Run agent
```cmd
cd momobot-agent
npm install
npm start
```

### Step 3: Look for
```
✅ Connected to MomoBot server
```

## 🔧 Troubleshooting Cheatsheet

| Problem | Solution |
|---------|----------|
| npm not recognized | Run: COMPLETE_SETUP.bat |
| Can't find dependencies | Run: npm install |
| Can't connect to server | npm start in server folder |
| Port 4000 in use | taskkill /F /IM node.exe |
| Don't know what's wrong | Run: DIAGNOSE_AGENT.ps1 |
| Agent keeps disconnecting | Check SERVER_URL in .env |
| Want desktop shortcut | Run: CREATE_SHORTCUTS.ps1 |

## ⚡ Command Line Cheatsheet

```cmd
# Install Node.js (if missing)
Visit: nodejs.org and download LTS

# Check versions
node --version    ← Should show v18+
npm --version     ← Should show 8+

# Setup agent
cd momobot-agent
npm install       ← Install once

# Run agent
npm start         ← Start daily

# Stop agent
Ctrl+C           ← Press in terminal

# Kill all Node processes
taskkill /F /IM node.exe

# Check what's using port 4000
netstat -ano | findstr :4000

# View logs
type logs\agent.log

# Run diagnostics (PowerShell)
powershell -ExecutionPolicy Bypass -File "DIAGNOSE_AGENT.ps1"

# Create shortcuts (PowerShell)
powershell -ExecutionPolicy Bypass -File "CREATE_SHORTCUTS.ps1"
```

## 🔐 Environment Variables (.env)

```env
Required:
  AGENT_API_KEY=...           ← Your API key
  AGENT_SECRET_KEY=...        ← Your secret
  SERVER_URL=http://localhost:4000

Optional:
  LOG_LEVEL=info              ← debug/info/warn/error
  HEARTBEAT_INTERVAL=30000    ← Milliseconds
  RECONNECT_DELAY=5000        ← Milliseconds
```

Edit: momobot-agent\.env

## 🚨 Emergency Commands

**Agent won't start?**
```cmd
npm install                   ← Reinstall dependencies
npm start                     ← Try again
```

**Everything stuck?**
```cmd
taskkill /F /IM node.exe     ← Kill all Node processes
npm start                     ← Start fresh
```

**Port conflicts?**
```cmd
netstat -ano | findstr :4000 ← Find what's using it
taskkill /PID <ID> /F        ← Kill the process
npm start                     ← Try again
```
