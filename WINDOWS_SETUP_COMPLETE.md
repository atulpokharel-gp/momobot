## 🚨 If Something Goes Wrong

### Step 1: Run Diagnostics
```powershell
powershell -ExecutionPolicy Bypass -File "DIAGNOSE_AGENT.ps1"
```

### Step 2: Check Common Issues

**"npm is not recognized"**
- Run: COMPLETE_SETUP.bat
- It auto-installs Node.js

**"Cannot find dependencies"**
- Run: npm install (in momobot-agent folder)
- Or use: START_AGENT.bat (auto-installs)

**"Connection refused"**
- Check server is running: npm start (server folder)
- Verify SERVER_URL in .env
- Check port 4000 is available

**"Port already in use"**
- Kill existing processes: taskkill /F /IM node.exe
- Or change port in .env

```cmd

# Stop agent

# Kill all Node processes
```
